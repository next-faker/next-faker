# Animal

Module to generate animal related entries.

* Bear
* Bird
* Cat
* Cetacean
* Cow
* Crocodilia
* Dog
* Fish
* Horse
* Insect
* Lion
* Rabbit
* Snake
* Type

## Bear

Returns a random bear species.

**Returns:** string

```ts
// bear function
animal.bear(): string

// import
import { useFaker } from 'next-faker';

const { animal } = useFaker();

// usage
animal.bear(); // => "American black bear"
animal.bear(); // 'Asian black bear'
```
