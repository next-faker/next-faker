## Building Number

Generates a random building number.

**Returns:** string

```ts
// buildingNumber function
address.buildingNumber(): string

// import
import { useFaker } from 'next-faker';
const { address } = useFaker();

// usage
address.buildingNumber(); // => "5786"
address.buildingNumber(); // '379'
```
